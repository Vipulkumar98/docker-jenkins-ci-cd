# Docker Jenkins CI/CD Project

This project demonstrates a simple CI/CD workflow using:

- GitHub
- Docker
- Docker Hub
- Jenkins
- Flask
- Ubuntu Linux

## Application
A simple Flask web application is containerized using Docker and automated using Jenkins Pipeline.

## CI/CD Workflow
1. Source code stored in GitHub
2. Jenkins pulls source code
3. Docker image is built from Dockerfile
4. Image is pushed to Docker Hub
5. Docker container is deployed
